# User Story: 應用自訂評分方案

## 基本信息
- **ID**: US-004
- **版本**: Release_2
- **狀態**: Pending
- **優先級**: Medium

## User Story 描述

**作為** 授課教師

**我想要** 能夠定義和應用不同的評分方案（如平績權重、加權計分等），根據不同考核組件計算最終成績

**以便於** 靈活適應不同課程的評分需求，確保評分的公平性和公正性

## 接受標準

1. [ ] 支持創建多個評分方案模板（如平均分、加權分等）
2. [ ] 支持設置各個考核組件的權重（如平時20%、期溫30%、期末50%）
3. [ ] 支持多種評分計算規則：簡單平均、加權平均、最高分優先等
4. [ ] 支持預設常見評分方案（如百分制、五級制等）
5. [ ] 提供方案預覽功能，顯示應用前後的成績變化
6. [ ] 支持保存多個評分方案並在課程間復用
7. [ ] 可應用和撤銷評分方案更改
