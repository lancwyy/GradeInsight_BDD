# User Story: 查看學生個人成績詳情

## 基本信息
- **ID**: US-006
- **版本**: Release_2
- **狀態**: Pending
- **優先級**: Medium

## User Story 描述

**作為** 授課教師

**作為** 課程助教

**我想要** 能夠點擊查看任意學生的詳細成績信息，包括各次考試成績、作業成績、參與度等組件，以及變化趨勢

**以便於** 更深入地了解學生的學習進度，識別學生的強項和弱項

## 接受標準

1. [ ] 顯示學生的基本信息（學號、姓名、班級等）
2. [ ] 列出所有評分組件及其分數
3. [ ] 展示成績在班級中的排名
4. [ ] 提供成績變化趨勢圖表（如多次考試的分數走勢）
5. [ ] 顯示與班級平均分的對比
6. [ ] 可查看詳細的評分說明和備註
7. [ ] 支持打印或導出個人成績單
