# User Story (已修正): 編輯單一學生成績

## 基本資訊
- **ID**: US-008-revised
- **版本**: Release_2
- **狀態**: Pending
- **優先順序**: High

## User Story 描述

**作為** 授課教師

**作為** 課程助教

**我想要** 能夠快速編輯和修改單一學生的成績，並記錄修改歷史

**以便於** 修正輸入錯誤或根據復查結果調整成績

## 接受標準

1. [ ] 支援點選成績儲存格進行快速編輯
2. [ ] 編輯前要求輸入修改原因（如"復查發現計分錯誤"）
3. [ ] 支援修改單一成績元件（如某次考試、某項作業）
4. [ ] 修改後自動重新計算最終成績
5. [ ] 記錄修改的時間、修改人、修改原因
6. [ ] 提供撤銷功能回到上一個版本
7. [ ] 所有修改需要稽核紀錄記錄
