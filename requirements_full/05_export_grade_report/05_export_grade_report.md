# User Story: 匯出成績報告

## 基本資訊
- **ID**: US-005
- **版本**: Release_1
- **狀態**: In Progress
- **優先順序**: High

## User Story 描述

**作為** 授課教師

**作為** 課程助教

**我想要** 能夠產生和匯出各種格式的成績報告（如PDF、Excel等），包括個人報告和班級報告

**以便於** 提交給學校教務系統、發給學生，或用於教學檔案歸檔

## 接受標準

1. [ ] 支援匯出格式：PDF、Excel、CSV
2. [ ] 支援產生班級總成績表報告
3. [ ] 支援產生個人成績單
4. [ ] 支援自訂報告內容範本
5. [ ] 報告中包含統計資訊：平均分、及格率、優良率等
6. [ ] 支援批次匯出所有學生的成績單
7. [ ] 報告可包含條碼或QR碼便於識別
8. [ ] 支援設定匯出資料的隱私級別

## 相關資訊
