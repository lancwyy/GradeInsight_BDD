# User Story: 導出成績報告

## 基本信息
- **ID**: US-005
- **版本**: Release_1
- **狀態**: In Progress
- **優先級**: High

## User Story 描述

**作為** 授課教師

**作為** 課程助教

**我想要** 能夠生成和導出各種格式的成績報告（如PDF、Excel等），包括個人報告和班級報告

**以便於** 提交給學校教務系統、發給學生，或用於教學檔案歸檔

## 接受標準

1. [ ] 支持導出格式：PDF、Excel、CSV
2. [ ] 支持生成班級總成績表報告
3. [ ] 支持生成個人成績單
4. [ ] 支持自訂報告內容範本
5. [ ] 報告中包含統計信息：平均分、及格率、優秀率等
6. [ ] 支持批量導出所有學生的成績單
7. [ ] 報告可包含條形碼或QR碼便於識別
8. [ ] 支持設置導出數據的隱私級別
