# User Story (已修正): 成績自動調整規則

## 基本信息
- **ID**: US-009-revised
- **版本**: Release_2
- **狀態**: Pending
- **優先級**: High

## User Story 描述

**作為** 授課教師

**我想要** 能夠定義和應用成績自動調整規則，例如根據出勤率自動扣分或獎勵

**以便於** 自動應用評分政策，提高評分的一致性和效率

## 接受標準

1. [ ] 支持定義調整規則（如"缺勤3次以上扣5分"）
2. [ ] 支持多種規則類型：扣分規則、獎勵規則、比例調整規則
3. [ ] 應用規則前提供預覽，顯示會影響哪些學生和變化範圍
4. [ ] 規則應用是可追蹤的，記錄哪個規則進行了調整
5. [ ] 支持撤銷特定規則的應用
6. [ ] 規則應用後不應再次自動應用（防止重複）
7. [ ] 支持保存和復用規則模板

## 相關資訊

**衝突**: 無

**依賴**: US-001（匯入學生成績）、US-004（應用自訂評分方案）

**備註**: 自動調整應得到人工確認，不應在未經授權的情況下自動執行
