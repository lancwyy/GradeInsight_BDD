# User Story (已修正): 成績自動調整規則

## 基本資訊
- **ID**: US-009-revised
- **版本**: Release_2
- **狀態**: Pending
- **優先順序**: High

## User Story 描述

**作為** 授課教師

**我想要** 能夠定義和套用成績自動調整規則，例如根據出勤率自動扣分或獎勵

**以便於** 自動套用評分政策，提高評分的一致性和效率

## 接受標準

1. [ ] 支援定義調整規則（如"缺勤3次以上扣5分"）
2. [ ] 支援多種規則類型：扣分規則、獎勵規則、比例調整規則
3. [ ] 套用規則前提供預覽，顯示會影響哪些學生和變化範圍
4. [ ] 規則套用是可追蹤的，記錄哪個規則進行了調整
5. [ ] 支援撤銷特定規則的套用
6. [ ] 規則套用後不應再次自動套用（防止重複）
7. [ ] 支援儲存和重複使用規則範本

## 相關資訊

**衝突**: 無

**依賴**: US-001（匯入學生成績）、US-004（套用自訂評分方案）

**備註**: 自動調整應得到人工確認，不應在未經授權的情況下自動執行
