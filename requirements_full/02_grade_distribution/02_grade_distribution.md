# User Story: 檢視成績分布

## 基本資訊
- **ID**: US-002
- **版本**: Release_1
- **狀態**: In Progress
- **優先順序**: High

## User Story 描述

**作為** 授課教師

**作為** 課程助教

**我想要** 能夠檢視整個班級的成績分布情況，包括直方圖、統計圖表等視覺化展示

**以便於** 快速了解班級整體的學習成果分布，識別成績段位分布是否合理

## 接受標準

1. [ ] 顯示成績分布的直方圖（縱軸為人數，橫軸為成績段位）
2. [ ] 支援自訂成績段位寬度（如5分、10分等）
3. [ ] 顯示关键統計指標：平均分、中位數、標準差、最高分、最低分
4. [ ] 支援按等級（A、B、C、D、F）分類統計
5. [ ] 支援按課程模組或考試類型篩選資料
6. [ ] 圖表檔案可匯出為PNG或PDF格式
